说明

在Java中要想实现动态代理，则需要java.lang.reflect.InvocationHandler接口和java.lang.reflect.Proxy类的支持

在此接口中只定义了一个invoke()方法，此方法有三个参数，其参数的含义如下：

  • Object proxy:被代理的对象
  • Method method:要调用的方法
  • Object args[]:方法调用时所需的参数
